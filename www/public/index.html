xmreuse

use /api/method




API methods:


pools/list

Look up a list of pools scraped by xmreuse



Parameters:

json boolean
Return JSON object (optional)

Return:

A list of pools, one per line or with more information if json is set


/pool/coinbase_outs

Look up a pool's coinbase outputs



Parameters:

pool string
Pool to look up

json boolean
Return JSON object (optional)

Return:

A list of pool's coinbase outputs, one per line or with more information if json is set


/pool/blocks

Look up a pool's blocks



Parameters:

pool string
Pool to look up

json boolean
Return JSON object (optional)

Return:

A list of pool's blocks, one per line or with more information if json is set
